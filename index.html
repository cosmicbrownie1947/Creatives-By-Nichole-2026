Project Overview

Reality Check is an interactive educational game designed to help users identify the difference between real images and AI-generated images. As artificial intelligence becomes more advanced, it is becoming increasingly difficult to tell what is real online. This project aims to build media literacy skills through an engaging guessing game.

Key Features

AI vs Real Game

Players are shown images and must decide whether they are AI generated or real.

Instant Feedback

If the player guesses incorrectly, the game provides an explanation showing how the image was created.

Score Tracking

Users play multiple rounds and receive a final score showing how well they can detect AI generated images.

Project Goals

  • Increase awareness of AI generated media
  • Teach users visual clues that reveal AI images
  • Create an engaging educational game
  • Promote digital media literacy
